# FinKit

FinKit 是為台灣使用者打造的理財輔助工具箱，提供預算、淨資產、緊急預備金、負債、資產配置、複利、定期定額、台股交易、配息、房貸、買租比較、FIRE、保險缺口、通膨、匯率、海外所得與 FCN 試算。

使用者可以用簡易本機登入建立個人檔，試算資料會依暱稱/Email 分開暫存在同一台裝置的瀏覽器 localStorage。這不是雲端帳號系統，不會跨裝置同步。

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
